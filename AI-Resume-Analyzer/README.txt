 #AI Resume Analyzer

An AI-powered Resume Analyzer built with **LangChain**, **Hugging Face**, **Streamlit**, and **Pydantic**. Upload a PDF resume to receive a structured AI analysis, including extracted information, resume score, strengths, weaknesses, and recommended job roles.

##  Features

- Upload PDF resumes
- Extract resume text automatically
- AI-powered resume analysis
- Structured output using Pydantic
- Resume scoring (0–10)
- Strengths & weaknesses detection
- Job role recommendations
- Interactive Streamlit interface

##  Tech Stack

- Python
- LangChain
- Hugging Face
- Streamlit
- Pydantic
- PyPDF
